# CodeAlpha Machine Learning Internship Tasks

## Overview

This repository contains the machine learning projects completed as part of the CodeAlpha Machine Learning Internship Program. The projects demonstrate the application of machine learning and deep learning techniques in finance, healthcare, and computer vision.
---------------------------------------------------------------------------------------------------------------------------------------------------------

## Projects Included

### 1. Credit Scoring Model

**Objective:**
Predict an individual's creditworthiness using financial and credit-related information.

**Technologies Used:**

* Python
* Pandas
* NumPy
* Scikit-Learn

**Features:**

* Data preprocessing
* Classification using Random Forest
* Creditworthiness prediction
* Model evaluation using Accuracy, Precision, Recall, F1-Score, and ROC-AUC

--------------------------------------------------------------------------------------------------------------------------------------------------------

### 2. Handwritten Character Recognition

**Objective:**
Recognize handwritten digits using deep learning techniques.

**Technologies Used:**

* Python
* TensorFlow
* Keras
* NumPy

**Features:**

* MNIST handwritten digit dataset
* Convolutional Neural Network (CNN)
* Image classification
* High-accuracy digit recognition

------------------------------------------------------------------------------------------------------------------------------------------------------------

### 3. Disease Prediction from Medical Data

**Objective:**
Predict disease outcomes using patient medical information.

**Technologies Used:**

* Python
* Scikit-Learn
* NumPy

**Features:**

* Breast Cancer Wisconsin Dataset
* Random Forest Classification
* Disease prediction
* Performance evaluation using standard metrics

## Internship

These projects were developed and submitted as part of the CodeAlpha Machine Learning Internship Program.
