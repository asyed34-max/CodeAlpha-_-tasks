# Disease Prediction from Medical Data

## Overview

This project is a machine learning-based disease prediction system developed using the Breast Cancer Wisconsin Dataset. The objective is to predict whether a tumor is benign or malignant based on medical measurements. The project demonstrates the application of machine learning techniques in the healthcare domain to assist in disease diagnosis and prediction.
The model is trained using the Random Forest Classifier algorithm and evaluated using various performance metrics such as Accuracy, Precision, Recall, and F1-Score.

## Objectives

* Predict disease outcomes using medical data.
* Apply machine learning classification techniques.
* Train and evaluate a predictive model.
* Demonstrate the use of AI in healthcare applications.

## Technologies Used

* Python
* Scikit-Learn
* NumPy
* Google Colab

## Dataset

Breast Cancer Wisconsin Dataset

The dataset contains medical measurements collected from breast cancer patients. These features are used to classify tumors into two categories:

* Benign (Non-Cancerous)
* Malignant (Cancerous)

## Machine Learning Algorithm

Random Forest Classifier

## Project Workflow

1. Load the Breast Cancer Dataset
2. Split Data into Training and Testing Sets
3. Train the Random Forest Model
4. Evaluate Model Performance
5. Predict Disease Status for New Samples
6. Compare Actual and Predicted Results

## Evaluation Metrics

The model performance is evaluated using:

* Accuracy Score
* Precision Score
* Recall Score
* F1 Score
* Confusion Matrix
* Classification Report

## Results

The trained model achieves high prediction accuracy and successfully classifies tumors as benign or malignant. The project also demonstrates real sample predictions using unseen test data.
