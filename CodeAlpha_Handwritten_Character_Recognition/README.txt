CREDIT SCORING MODEL

OVERVIEW
This project uses Machine Learning to predict whether a customer is creditworthy based on financial information such as age, income, loan amount, credit history, and existing debt. The model helps evaluate the risk of granting credit to an applicant.

OBJECTIVES
- Predict customer creditworthiness.
- Apply machine learning classification techniques.
- Evaluate model performance using standard metrics.
- Demonstrate the use of data-driven decision making in finance.

TECHNOLOGIES USED
- Google colab
- Python
- Pandas
- NumPy
- Scikit-Learn

DATASET FEATURES
- Age
- Income
- Loan Amount
- Credit History
- Existing Debt

TARGET VARIABLE
Creditworthy
1 = Eligible for Credit
0 = Not Eligible for Credit

MACHINE LEARNING MODEL
Random Forest Classifier

WORKFLOW
1. Data Collection
2. Data Preprocessing
3. Feature Selection
4. Train-Test Split
5. Model Training
6. Prediction
7. Model Evaluation

EVALUATION METRICS
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC Score
- Confusion Matrix

PROJECT FILES
- credit_scoring.py
- requirements.txt
- README.txt
- Credit_Scoring_Model.ipynb

RESULT
The model predicts whether a customer is creditworthy or not based on financial attributes and displays performance metrics after training.
